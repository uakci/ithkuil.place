ithkuil.place

Welcome! Attalú luyün! Attàwîl üpʰu! On this webpage, you will find assorted tools, resources, and APIs whose collective goal is to aid in learning and using the Ithkuilic languages.

This website is not affiliated with the creator of Ithkuil, John Quijada.

Got any questions? Want to learn one of the languages? We welcome you to join our Discord server and have a chat!
